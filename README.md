# YMIR Assignment


## About
- Idea of this project is to get all the texts from the uploaded image.
- For the recognistion of the text, google vision API is being used. 

### Project setup

```
npm install
```

### Compiles and hot-reloads for development


```
npm start
```

### Tech used

- ReactJS
- React Bootstrap
- Axios
- React file reader
